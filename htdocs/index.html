songtickerli

Ticker

Infos

Auf songtickerli findest du einen Songticker für das Radio RaBe in Bern. Der Songticker ist so gestaltet das er sich mit Javascript einfach in eine bestehende Seite integrieren lässt.

Der Songticker befindet sich noch in Entwicklung weswegen sich hier auf songticker.li noch einiges ändern wird. Er ist darum noch nicht wirklich für den Einsatz im Internet brauchbar.

Beispielcode

Javascript


<script type="text/javascript" src="http://songticker.li/rabe.js"></script>
<style>
  #songtickerli {
    border: 1px solid black;
  }
  .songtickerli-message {
    font-size: 90%;
  }
  .songtickerli-artist {
    font-weight: bolder;
    padding-right: 1em;
  }
  .songtickerli-title {
    font-size: 90%;
  }
</style>
<div id="songtickerli"/>
