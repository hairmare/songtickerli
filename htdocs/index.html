Infos

Auf songtickerli findest du einen Songticker für das Radio RaBe in Bern. Der Songticker ist so gestaltet das er sich einfach in eine bestehende Seite integrieren lässt.

Beispielcode

Javascript

<script type="text/javascript" src="http://songticker.li/rabe.js"></script>

<noscript>

 <a href="http://www.rabe.ch"><img src="http://songticker.li/rabe_gruen.png"/></a>

</noscript>

Apache

ProxyPass        /songtickerli http://songticker.li/data/rabe.ch/
ProxyPassReverse /songtickerli http://songticker.li/data/rabe.ch/
CacheEnable  mem /songtickerli
CacheMaxExpire 16

Lizenz

Falls du den Songticker auf deiner Homepage einbindest, solltest du mindestens RaBe Supporter sein. Viel schöner wäre es natürlich wenn du ein richtiges Mitglied wirst: RaBe Mitglied werden.
